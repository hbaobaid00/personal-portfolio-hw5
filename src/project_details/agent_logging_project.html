Agent Logging & Autonomy Framework Evaluation

During my internship at SkillWorld, I was responsible for exploring and evaluating several autonomous agent frameworks by configuring them, running task-oriented agents, and capturing detailed execution logs for analysis. Frameworks tested include Auto-GPT, MetaGPT, CrewAI, and SuperAGI‚Äîeach showcasing different approaches to reasoning, coordination, tool use, and long-horizon task completion.

Purpose of the Work

Rather than training models or building agents from scratch, this project focuses on:

  • Setting up multiple agent frameworks using Docker and Python
  • Configuring agents for different task profiles
  • Capturing and storing execution logs for comparison
  • Analyzing agent behavior, reasoning patterns, and tool usage

Frameworks Tested & Example Experiments

Auto-GPT

Deployed inside Docker to test autonomous goal pursuit and iterative reasoning loops. Logged agents included a mathematical calculator task, GitHub automation behaviors, and a Discord-based agent.

MetaGPT

Evaluated MetaGPT's software-engineering role simulation using the prompt: "Write a CLI snake game." Logs captured task decomposition and planning output from its multi-role agent orchestration.

CrewAI - Trip Planner

Configured a multi-agent trip planner that generated itinerary recommendations from San Diego to Switzerland. Logged versions show iterative reasoning, expert collaboration, and JSON-structured task breakdowns, ultimately recommending Interlaken.

SuperAGI

Used to execute a goal-driven research agent tasked with ranking top sightseeing countries and emailing results. Logs document its retrieval actions, file generation, and email delivery.

Logging Details

For every agent, file outputs were configured to capture:

  • Timestamps and tool invocations
  • Inter-agent dialogue where applicable
  • Final outputs and failure cases
  • Behavioral patterns observable across runs

Logs are aggregated in the project structure and used to compare autonomy frameworks, debugging styles, strengths, goals, and action sequencing.
